TortoiseGit + GitHub使用教程.md

安装TortoiseGit，首先需要安装mysysgit注册安装GitHubhttp://www.360doc.com/content/13/0510/15/7635_284404426.shtml全部安装好后，可以填写账号密码登陆TortoiseGit可以直接commit到GitHub，但是不能sync使用PuTTYgen可以生成公钥，通过鼠标不停的移动，还可以设置Key passphare，TortoiseGit pull的时候需要输入，save private key,保存的是私钥，软件上显示的是公钥，复制公钥 将public key 的内容添加到GitHub中，通过网页操作seting->SSH keys设置TortoiseGit ->settings->Git->Remote设置url和Pytty之后就可以使用TortoiseGit pull直接上传更新到github了老是pull，点错了呀混蛋，是push
