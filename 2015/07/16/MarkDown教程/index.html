MarkDown教程

日志

WHAT

markdown是一种轻量级的标记语言。常用的标记符号不超过十个，使用的时候简单易懂就行了，不需要太多花哨

优点

  • 专注你的文字内容而不是排版样式
  • 轻松导出HTML、PDF 和本身的 md 文件
  • 纯文本编辑，适用于所有的编辑器和字处理软件
    * 可读，直观。适合所有人的写作语言

工具

  • sublime
  • wiz
  • mou(mac)

HOW

  • 创始人 John Gruber 的 MarkDown 语法说明
  • MarkDown 中文语法说明
  • 认识与入门MarkDown

标题

只用在一行的文字前面加1到6个#即可，分别表示1到6级标题
#标题
##标题
###标题

列表

在文字前面加上-或者*，表示无序列表，加上1. 2. 3.表示有序列表，符号与文字中间要加上一个空格
有序列表前面的数字可以随便写并不影响生成的顺序

引用

如果要引入一段句子，可以在前面加上>即可

图片与链接

[baidu](http://www.baidu.com title )
![image](htt://image.com title)
插入图片和链接也有其他的方式比如说：
[baidu][]
[baidu]: http://www.baidu.com "title"

推荐CloudApp 的服务，上传图片生成URL即可

粗体与斜体，删除线

前后加一个*或者-表示斜线
前后加三个*或者-表示粗体
前后加一个~表示删除线

代码段

前面加4个空格或者一个制表符，即可生成代码段
或者一段代码前后都加上3个`

分割线

连续的三个*或者-即可生成一条华丽的分割线

总结

以上是基本的MarkDown使用方法，喏，接触之前感觉好厉害好厉害，好难好难，这么高级的东西要学很久吧！
结果，学习加整理加差不多已经熟练起来不过半天时间。Just do it
